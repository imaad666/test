# Project Overview
This project provides a basic React application with API connectivity and database schema.
It includes pages for the main application and a new outreach page.

## New Features
- Added new page for outreach with minimal dependencies

## Project Structure
- src: Source code for the React application
- src/App.jsx: Main application component
- src/OutreachPage.jsx: New page component for outreach
- src/helpers: Helper functions for API routes and other utilities
- src/helpers/routes.js: Routes for the application
- server.js: Server-side code for API connectivity
- schema.sql: Database schema

## Running the Application
1. Install dependencies
2. Start the server
3. Open the application in a web browser

## Testing
- Run tests using the test files provided
